# Awesome Buttons

CSSアニメーションを使ったボタンのサンプルです。ご自由にお使いください。

▼プレビューをご覧いただけます

[プレビューを見る](https://kutaro-smile.github.io/awesome-buttons/index.html)

## スタイルの場所

ボタンだけのスタイルを記述したCSS、Sassファイルを、それぞれ用意しております。下記ファイルをご覧ください。

### CSSのスタイル

* css > button.css

### Sassのスタイル

* scss > button.scss
